### 🚀 What I Do

- 🗺️ **Spatial & Search Systems**: Raw PostGIS spatial SQL, custom search infrastructure with hybrid retrieval, FastAPI with background job queues.
- 🔐 **Backend Architecture**: RBAC middleware, Redis backed rate limiting with atomic Lua scripts, PostgreSQL concurrency handling with SELECT FOR UPDATE SKIP LOCKED.
- ☁️ **Cloud & DevOps**: Docker image optimization, k3s deployment on Azure, CI/CD pipelines via GitHub Actions.
- 🌐 **Full Stack Delivery**: Next.js 14 frontends wired to real backends, PWAs, and real time systems with Supabase.
